abstract fun useLegacyReceiverNoiseRemoval(useLegacy: Boolean)

API to enable legacy receiver side noise removal. This will disable the new receiver side speech enhancement. Disable this to use new receiver side speech enhancement. This will disable the legacy receiver side noise removal. By default useLegacy is false. This setting is not persisted.

Since

3.14.0

Parameters

useLegacy

: true to enable legacy receiver side noise removal, false to disable
